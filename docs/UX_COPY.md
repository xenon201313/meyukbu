# UX 및 한국어 카피

## 브랜드

- 로고 텍스트: `메력서`
- 보조 설명: `메이플 파티 구직용 캐릭터 이력서`
- 슬로건: `스펙을 다시 쓰지 말고, 한 장으로 증명하세요.`

## 랜딩

### Hero

- 제목: `캐릭터명 하나로 만드는 파티 구직 메력서`
- 설명: `API 조회 정보와 파티 경험·가능 시간을 한 장에 정리하고, QR로 데이터 기준 시각과 원본을 확인하세요.`
- 검색 placeholder: `캐릭터명을 입력하세요`
- CTA: `메력서 만들기`
- 보조 안내: `캐릭터명을 검색하면 메력서 작성에 필요한 공개 정보를 불러옵니다.`

### 신뢰 고지

`API 조회값, 서비스 계산값, 작성 내용을 구분해 표시합니다.`

## 환산·보스 배율 참고

- 안내: `메력서는 환산과 보스 배율을 자동으로 가져오거나 임의로 계산하지 않습니다. 확인한 값은 직접 입력할 수 있으며, 메력서에는 작성 내용으로 표시됩니다.`
- 외부 확인: `MapleScouter에서 환산·보스 배율 확인`
- 환산 placeholder: `예: 110,650`
- 보스 배율 placeholder: `예: 412.5`
- 보조 안내: `보스 배율에는 % 기호를 제외한 숫자만 입력해 주세요.`

## 보스 구분

- WEEKLY: `주간 보스`
- MONTHLY: `월간 보스`
- 안내: `먼저 주간 또는 월간을 고른 뒤, 난이도가 포함된 지원 보스를 목록에서 선택하세요.`
- 보조 설명: `카드 이미지는 선택을 돕는 자체 제작 일러스트이며, 게임 공식 보스 이미지가 아닙니다.`
- 인원 안내: `선택한 보스는 최대 {n}인격까지 입장할 수 있습니다.`
- 스우 제한: `스우는 최대 2인격까지 선택할 수 있습니다.`
- 3인 제한: `최초의 대적자, 림보, 발드릭스, 찬란한 흉성, 유피테르는 최대 3인격까지 선택할 수 있습니다.`

## 참여 시간 방식

- SCHEDULED: `요일·시간 지정`
- NEGOTIABLE: `요일·시간 협의 가능`
- FLEXIBLE: `요일·시간 무관`
- 협의 안내: `요일과 시간은 파티와 협의할 수 있습니다.`
- 무관 안내: `요일과 시간에 관계없이 참여할 수 있습니다.`

## 전투력과 최종 능력치

- 섹션 제목: `API 조회 전투력 상세`

- 조회 불가: `이 항목은 현재 API에서 불러오지 못했습니다.`

## 출처 배지

- NEXON_API: `API 조회`
- SERVICE_CALCULATED: `서비스 계산`
- USER_PROVIDED: `작성 내용`
- SERVICE_OBSERVED: `서비스 관측`

## 신선도

- fresh: `최근 조회`
- stale: `업데이트 확인 권장`
- expired: `갱신 필요`

stale 안내:
`이 메력서의 API 데이터는 최근 24시간 이내 값이 아닙니다. 최신 정보를 확인하려면 갱신하세요.`

expired 안내:
`API 데이터가 오래되어 공개가 제한되었습니다. 작성자가 갱신하면 다시 확인할 수 있습니다.`

## 오류

- 빈 이름: `캐릭터명을 입력해 주세요.`
- 찾을 수 없음: `해당 캐릭터를 찾지 못했습니다. 이름을 다시 확인해 주세요.`
- rate limit: `조회 요청이 많습니다. 잠시 후 다시 시도해 주세요.`
- maintenance: `현재 게임 데이터 조회가 원활하지 않습니다. 잠시 후 다시 시도해 주세요.`
- partial: `일부 능력치 또는 장비 정보를 불러오지 못했지만 기본 정보로 계속 작성할 수 있습니다.`
- no API key demo: `현재 데모 데이터로 표시 중입니다.`

## 나의 이력서

- 메뉴: `나의 이력서`
- 제목: `나의 이력서`
- 설명: `보스별로 저장한 메력서를 확인하고, 기존 이력서를 유지한 채 새 이력서로 복제할 수 있습니다.`
- 보스 필터: `희망 보스`
- 카드 동작: `메력서 열기`, `수정`, `새 메력서로 저장`
- 빈 상태 제목: `저장된 메력서가 없어요.`
- 빈 상태 안내: `이 목록은 로그인 대신, 메력서를 만들 때 이 브라우저에 저장한 편집 권한을 기준으로 표시합니다. 다른 기기, 시크릿 창 또는 쿠키를 지운 브라우저에서는 이전 이력서를 확인할 수 없어요.`
- 오류 안내: `잠시 후 다시 열어 보거나, 생성했던 브라우저인지 확인해 주세요.`

## 메숭이 체온

- 섹션 제목: `메숭이 체온 · 동행 기록`
- 안내: `함께한 파티원이 남긴 기명 태그 기록입니다. 점수나 순위는 표시하지 않습니다.`
- 비보증 고지: `서비스는 실제 파티 참여나 게임 계정 소유권을 보증하지 않습니다.`
- 빈 상태: `아직 공개된 동행 기록이 없습니다.`
- 링크 발급: `동행 확인 링크 만들기`
- 링크 안내: `링크는 한 번만 사용할 수 있고 7일 뒤 만료됩니다. 실제 함께한 파티원에게만 전달해 주세요.`
- 평가자 안내: `본인이 관리하는 공개 메력서의 링크와 이 브라우저의 편집 권한이 필요합니다.`
- 태그 선택: `동행 태그를 1~3개 선택해 주세요.`
- PUNCTUAL: `약속 시간 준수`
- PREPARED: `공략 준비`
- COMMUNICATIVE: `원활한 소통`
- PERSISTENT: `재도전 적극성`
- FAIR_LOOT: `공정한 분배`
- 자기 평가 오류: `같은 캐릭터의 메력서에는 동행 기록을 남길 수 없습니다.`
- 중복 오류: `이 캐릭터는 이 메력서에 이미 동행 기록을 남겼습니다.`
- 초대 오류: `동행 확인 링크가 만료됐거나 이미 사용되었습니다.`

## 고지

- `Data based on NEXON Open API`
- `본 서비스는 NEXON의 공식 제휴 또는 인증 서비스가 아닙니다.`
- `작성 내용은 작성자가 직접 제공한 정보입니다.`
- `서비스 관측 이력은 실제 가입·탈퇴 시점과 다를 수 있습니다.`

## 디자인 원칙

- 모바일 우선
- 이력서의 정돈된 구조와 게임 프로필의 개성을 결합
- 캐릭터 이미지는 강조하되 핵심 정보보다 크게 지배하지 않음
- 숫자는 4~6개만 크게 표시
- 공유 이미지에는 장비 전체 옵션을 넣지 않고, 검증 페이지에서만 상세를 펼친다.
- 전투력은 API 원값으로만 표시하고 서비스 계산값처럼 꾸미지 않는다.
- 출처는 badge, icon, section label 중 최소 두 방식으로 구분
- 주간·월간 보스 카드에는 자체 생성한 추상 아트만 사용하며, 특정 공식 보스 초상으로 보이게 만들지 않는다.
- 작은 카카오톡/Discord 미리보기에서도 지원 보스·역할·캐릭터명이 읽혀야 함
- 기본 테마는 밝은 종이색 배경, 명확한 경계, 높은 대비
- 특정 직업/성별/장비를 가치 판단하는 카피 금지
