## 브랜드

- 로고 텍스트: `메력서`
- 보조 설명: `메이플 파티 구직용 캐릭터 이력서`
- 슬로건: `스펙을 다시 쓰지 말고, 한 장으로 증명하세요.`

## 랜딩

### Hero

- 제목: `캐릭터명 하나로 만드는 파티 구직 메력서`
- 설명: `API 조회 정보와 파티 경험·가능 시간을 한 장에 정리하고, QR로 데이터 기준 시각과 원본을 확인하세요.`
- 검색 placeholder: `캐릭터명을 입력하세요`
- CTA: `메력서 만들기`
- 보조 안내: `캐릭터명을 검색하면 메력서 작성에 필요한 공개 정보를 불러옵니다.`

## 환산·보스 배율 참고

- 안내: `메력서는 환산과 보스 배율을 자동으로 가져오거나 임의로 계산하지 않습니다. 환산은 아래에, 보스 배율은 각 희망 보스에 직접 기록하며 메력서에는 작성 내용으로 표시됩니다.`
- 외부 확인: `MapleScouter에서 환산·보스 배율 확인`
- 환산 placeholder: `예: 110,650`
- 보스 배율 placeholder: `예: 412.5`
- 보조 안내: `보스 배율에는 % 기호를 제외한 숫자만 입력해 주세요.`

## 희망 보스 묶음

- WEEKLY: `주간 보스`
- MONTHLY: `월간 보스`
- 안내: `같은 파티로 함께 갈 보스를 최대 6개까지 묶어 선택하세요.`
- 배율 안내: `보스 배율은 각 보스마다 % 기호 없이 숫자로 입력해 주세요.`
- 인원 안내: `선택한 보스 묶음은 최대 {n}인격까지 입장할 수 있습니다.`
- 스우 제한: `스우는 최대 2인격까지 선택할 수 있습니다.`
- 3인 제한: `최초의 대적자, 림보, 발드릭스, 찬란한 흉성, 유피테르는 최대 3인격까지 선택할 수 있습니다.`

## 신선도

- fresh: `최근 조회`
- stale: `업데이트 확인 권장`
- expired: `갱신 필요`

stale 안내:
`이 메력서의 API 데이터는 최근 24시간 이내 값이 아닙니다. 최신 정보를 확인하려면 갱신하세요.`

expired 안내:
`API 데이터가 오래되어 공개가 제한되었습니다. 작성자가 갱신하면 다시 확인할 수 있습니다.`

## 메붕이 온도

- 섹션 제목: `메붕이 온도`
- 게이지 제목: `현재 메붕이 온도`
- 기본 온도: `36.5℃`
- 안내: `익명 설문으로 반영된 파티 경험 지표입니다. 새 메력서를 만들어도 같은 캐릭터의 온도는 유지됩니다.`
- 이력서/PNG 표기: `메붕이 온도 · 36.5℃ · 익명 설문 0건`
- 비보증 고지: `서비스는 실제 파티 참여나 게임 계정 소유권을 보증하지 않습니다.`
- 빈 상태: `아직 반영된 익명 설문이 없습니다. 기본 메붕이 온도 36.5℃로 표시됩니다.`
- 링크 발급: `메붕이 온도 설문 링크 만들기`
- 링크 안내: `링크는 한 번만 사용할 수 있고 7일 뒤 만료됩니다. 실제 함께한 파티원에게만 전달해 주세요.`
- 익명 안내: `닉네임, 메력서 주소, 연락처를 입력하지 않는 익명 설문입니다. 설문자 신원과 개별 답변은 공개되지 않습니다.`
- 문항 1: `해당 파티원과의 보스 경험은 어땠습니까?`
- 문항 2: `해당 파티원의 숙련도는 만족스러웠습니까?`
- 문항 3: `시간 약속을 잘 지켰습니까?`
- 5점 선택: `매우 불만족 (-2℃)`, `불만족 (-1℃)`, `보통 (0℃)`, `만족 (+1℃)`, `매우 만족 (+2℃)`
- 시간 선택: `아니다 (-1℃)`, `그렇다 (+1℃)`
- 제출: `익명 설문 제출하기`
- 초대 오류: `설문 링크가 만료됐거나 이미 사용되었습니다.`

## 고지

- `Data based on NEXON Open API`
- `본 서비스는 NEXON의 공식 제휴 또는 인증 서비스가 아닙니다.`
- `작성 내용은 작성자가 직접 제공한 정보입니다.`
- `서비스 관측 이력은 실제 가입·탈퇴 시점과 다를 수 있습니다.`

## 디자인 원칙

- 모바일 우선
- 이력서의 정돈된 구조와 게임 프로필의 개성을 결합
- 캐릭터 이미지는 강조하되 핵심 정보보다 크게 지배하지 않음
- 숫자는 4~6개만 크게 표시
- 공유 이미지에는 장비 전체 옵션을 넣지 않고, 검증 페이지에서만 상세를 펼친다.
- 전투력은 API 원값으로만 표시하고 서비스 계산값처럼 꾸미지 않는다.
- 출처는 badge, icon, section label 중 최소 두 방식으로 구분
- 주간·월간 보스에는 사용자가 제공·사용을 허가한 Maple Trackers 일러스트만 보스와 1:1로 연결하고, 잘리지 않게 `object-contain`으로 표시한다.
- 작은 카카오톡/Discord 미리보기에서도 지원 보스·역할·캐릭터명이 읽혀야 함
- 기본 테마는 밝은 종이색 배경, 명확한 경계, 높은 대비
- 특정 직업/성별/장비를 가치 판단하는 카피 금지
